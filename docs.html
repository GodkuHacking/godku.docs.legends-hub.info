Godku Commands

Command Description
!invert Use this command to manually edit the file.
!14stars Make all your characters 14 stars.
!Zenkai Make all zenkai characters zenkai 7.
!Soulboost Max the soulboost of all your characters.
!Artboost Max the art boost to 99 for all the characters.
!Costume Give you the alternate color of all the characters.
!Equip Choose between absurd equipment power or maxed equipment with official stats (Second option is in work in progress).
!Level Change the level of all the characters (Only in friendly battle).
!14,7 Make the commands !14stars and !Zenkai at the same time.
!AllBoost Make the commands !Soulboost and !Artboost at the same time.
!Allmods Make the commands !14,7 and !Allboost at the same time.
!Ultimate Make the commands !Allmods and !Costume at the same time.
!Visual Give you all the characters in the game (WARNING: Like the command name said it's only visual you are not able to play the given character!).
!Team Edit your team through the file (Pair it with !Visual).
!Clone Put a wanted character in a team 6 times.
!Shallot Edit shallot costume, special move or transformation even if you don't have them.
!!background Changes the background of your game.
!Skip_tuto Skip the king kai tutorial on a new account.
!Infinite_CC Give you infinite CC trust.
!Sleeve Let you change the art sleeve (Id can be found on the 98c4 channel in the server or with the !Ids command).
!Ids Print the ids of all the sleeve.
!Apk Use this command to get the link to the different godku mod APK.
